# AI-powered Hand Protection System 🛡️

![Python](https://img.shields.io/badge/Python-3.13-blue.svg)

This concept is for a smart system to prevent hand jamming in inlet door mechanism of ARCS. Traditional hand protection system has been carried out by both of contact type or contactless type. But the common Issue with these method is that sensor must be within a certain range of the target object. Sensing contact force is a typical method with contactless type. However, this method still has a potential risk to lead to a weak or certain level of injury and this approach also has problem such as strict sensor surface management or exceptional situation of errors caused by foreign objects.
To tackle this above issue and risk, AI-Driven Hand Projection is proposed. A color which is easy to be distinguished or identified is adapted for a target object for AI visual model. Yellow tape will be attached around the profile of the inlet hole and a camera installed at the top of the inlet hole will tracks the tape’s size and position of the inlet in real time. Once any object hide the tape area partially, alarm signal will be generated and reopen the inlet door to avoid jamming it in between door mechanism.
